## 1. Системные требования

| Компонент | Минимум | Рекомендуется |
|-----------|---------|---------------|
| CPU | 2 ядра | 4 ядра |
| RAM | 4 ГБ | 8 ГБ |
| Диск | 10 ГБ SSD | 20 ГБ SSD |
| ОС | Linux / macOS / Windows 10+ | Ubuntu 22.04 LTS |
| Docker | 24.0+ | 25.0+ |
| Docker Compose | v2.20+ | v2.27+ |
| Git | 2.30+ | Latest |

### Сетевые порты
| Порт | Сервис | Назначение |
|------|--------|------------|
| 80 | Nginx | HTTP (редирект на HTTPS) |
| 443 | Nginx | HTTPS (основной доступ) |
| 8080 | Spring Boot | REST API + Swagger UI |
| 5432 | PostgreSQL | База данных (внутренняя сеть Docker) |
