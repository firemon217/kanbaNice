## COCOMO II — Базовая модель

**Тип проекта:** Semi-detached (полуотдельный)  
Используется для систем с умеренной степенью новизны и средним опытом команды.

### Формулы

```
E = a × (KSLOC)^b    — усилие (человеко-месяцы)
D = c × E^d          — продолжительность (месяцы)
P = E / D            — количество человек
```

### Коэффициенты (Semi-detached)

| Параметр | Значение |
|----------|---------|
| a | 3.0 |
| b | 1.12 |
| c | 2.5 |
| d | 0.35 |

### Расчёт

```
KSLOC = 5.15 (тысяч строк)

E = 3.0 × (5.15)^1.12
E = 3.0 × 6.27
E ≈ 18.8 человеко-месяца

D = 2.5 × (18.8)^0.35
D = 2.5 × 3.46
D ≈ 8.65 месяца

P = E / D = 18.8 / 8.65 ≈ 2.2 человека
```

### Итоговые оценки COCOMO II

| Метрика | Значение |

| Параметр | COCOMO оценка | Факт (курсовой) |

| Продолжительность | 8.65 мес | ~1 мес (интенсивная разработка) |

> **Вывод:** COCOMO даёт оценку для промышленной разработки с документацией и тестами. Курсовой проект выполнен в ускоренном режиме с фокусом на реализацию функционала.
