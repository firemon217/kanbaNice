# Руководство пользователя — kanbaNice

## 1. Введение

**kanbaNice** — система управления проектами по методологии Kanban для команд.  
Доступна в двух вариантах: веб-браузер и десктопное приложение (Windows/Linux/Mac).

---

## 2. Начало работы

### 2.1. Регистрация

1. Откройте приложение в браузере: `http://localhost`
2. Нажмите **«Зарегистрироваться»**
3. Введите:
   - Имя
   - Имя пользователя (логин)
   - Email
   - Пароль (минимум 6 символов)
4. Нажмите **«Создать аккаунт»**

**Альтернатива:** Нажмите **«Войти через Google»** для входа через аккаунт Google.

### 2.2. Вход в систему

1. Введите имя пользователя и пароль
2. Нажмите **«Войти»**

### 2.3. Восстановление пароля

1. На странице входа нажмите **«Забыли пароль?»**
2. Введите ваш email
3. Проверьте почту — придёт письмо со ссылкой сброса
4. Перейдите по ссылке, введите новый пароль

---

## 3. Управление компанией

> **Только для пользователей с ролью ADMIN**

### 3.1. Создание компании

1. Перейдите в раздел **«Компания»** через боковое меню
2. Введите название компании
3. Нажмите **«Создать»**

### 3.2. Добавление сотрудников

1. В разделе **«Компания»** найдите блок «Сотрудники»
2. Введите email нового сотрудника
3. Нажмите **«Добавить»**  
   *Сотрудник должен быть зарегистрирован в системе заранее*

### 3.3. Удаление сотрудника

1. В списке сотрудников нажмите иконку удаления рядом с именем
2. Подтвердите действие

---

## 4. Управление проектами

### 4.1. Создание проекта

1. Перейдите в раздел **«Проекты»**
2. Нажмите **«Создать проект»**
3. Введите название проекта
4. Нажмите **«Создать»**

### 4.2. Добавление участников в проект

1. Откройте нужный проект
2. Перейдите на вкладку **«Участники»**
3. Выберите сотрудника из списка и нажмите **«Добавить»**

### 4.3. Удаление проекта

1. Откройте проект
2. Нажмите **«Удалить проект»**
3. Подтвердите удаление

---

## 5. Работа с канбан-досками

### 5.1. Создание доски

1. Откройте нужный проект
2. Нажмите **«Добавить доску»**
3. Введите название (например: «Sprint 1», «Бэклог»)
4. Нажмите **«Создать»**

### 5.2. Переименование доски

1. Нажмите на иконку редактирования рядом с названием доски
2. Введите новое название
3. Сохраните

### 5.3. Удаление доски

1. Нажмите иконку удаления на доске
2. Подтвердите — **все задачи на доске будут удалены**

---

## 6. Работа с задачами

### 6.1. Создание задачи

1. В рамках нужной доски нажмите **«+ Добавить задачу»**
2. Введите заголовок и описание
3. Нажмите **«Создать»**

Задача создаётся со статусом **TODO**.

### 6.2. Изменение статуса задачи

1. Нажмите на задачу
2. Измените статус с **TODO** на **DONE**
3. Сохраните изменения

### 6.3. Редактирование задачи

1. Нажмите на задачу для открытия
2. Измените заголовок или описание
3. Нажмите **«Сохранить»**

### 6.4. Удаление задачи

1. Откройте задачу
2. Нажмите **«Удалить»**

---

## 7. Профиль пользователя

1. Нажмите на своё имя в боковом меню → **«Профиль»**
2. Доступные действия:
   - Изменить имя и email
   - Сменить пароль
   - Удалить аккаунт

---

## 8. Десктопный клиент (JavaFX)

Все функции идентичны веб-версии. Запуск:
```bash
cd desktop
./mvnw javafx:run
```
Введите URL сервера (по умолчанию: `http://localhost:8080`), затем войдите в систему.
